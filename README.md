## MIC-1

Esse repositório armazena o projeto MIC-1, das disciplinas Arquitetura e Organização de Computadores I e II (AOC I/AOC II). Nele está contido a implementação lógica da Unidade Lógica Aritmética e a Implementação do Banco de Registradores.
O software utilizado foi o Quartus II 13.1 (64 bits). O projeto possui suporte para compilação nas famílias CYCLONE II e CYCOLNE III.
